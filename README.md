# Whom Should I Root For?

Live site:

- <https://andrewburkard.com/whom-should-i-root-for/>

This is a static web app for comparing a March Machine Learning Mania submission
against a benchmark and showing which remaining results help your Brier score
the most.

If you need to refresh the bundled data locally, rebuild it from
`/Users/aburkard/fun/madness_pyro` with:

```bash
uv run python scripts/build_data.py
```
